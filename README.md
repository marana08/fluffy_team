# 🐾 Хатинка лапок

## 📝 Про проєкт

"Хатинка лапок" — односторінковий сайт притулку для тварин, реалізований
студентами GoIT у рамках навчального курсу Full Stuck development.

Проєкт створено для демонстрації роботи з REST API, реалізації адаптивного
дизайну, модальних вікон, динамічного рендерингу карток, фільтрації та
пагінації.

---

## 🐾 Опис проєкту

Односторінковий сайт притулку для тварин. Тут можна:

- обрати улюбленця для адопції;
- ознайомитися з вимогами до майбутніх опікунів;
- почитати відгуки тих, хто вже подарував дім хвостатим друзям.

Проєкт створено як навчальний, але з реальною метою — показати, що турбота та
любов змінюють життя.

---

## ✨ Функціональність

- 🐶 **Вибір тварини** — перегляд карток улюбленців із фото та описом.
- 💬 **Відгуки** — історії тих, хто вже подарував дім хвостатим друзям.
- ❓ **FAQ** — відповіді на поширені запитання про процес адопції.
- 📩 **Модальне вікно заявки** — проста форма для знайомства з обраною твариною.
- 🏠 **Про нас** → замінено на **Про команду**.

## 🛠️ Технології та інструменти

- 🌐 **HTML** — семантична розмітка сторінки.

## 📱 Адаптивність та доступність

### 📐 Адаптивність

- 📏 **Mobile**: верстка гумова від `320px`, стає адаптивною з `375px`.
- 📏 **Tablet**: адаптивна верстка з `768px`.
- 📏 **Desktop**: адаптивна верстка з `1440px`.

> ⚠️ На мобільних пристроях (до 375px) сайт відображається у гумовому форматі,
> без повної адаптації.

### ♿ Доступність

- 🔖 **aria-label** — для опису кнопок та інтерактивних елементів.
- 🗂️ **role** — визначення ролей елементів (навігація, кнопка, діалог).
- 🎙️ **aria-live** — для повідомлень та нотифікацій.
- ⌨️ **Клавіатурна навігація** — фокус та доступність основних елементів через
  `tab`.
- 🖼️ **alt-тексти** — для всіх зображень тварин та іконок.
- 🎨 **Контрастність кольорів** — відповідність базовим вимогам WCAG.
- 🧭 **aria-expanded / aria-controls** — для акордеонів у FAQ.
- 📋 **aria-modal** — для модальних вікон заявки.
